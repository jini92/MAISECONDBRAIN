# Tistory Representative Image Strategy - MAISECONDBRAIN

- **Visual style**: title-only cover with a clean knowledge-graph / PKM mood. No screenshot collage unless the post is explicitly a UI showcase.
- **Palette**: charcoal `#111827`, violet `#7C3AED`, teal `#14B8A6`, soft gray `#E5E7EB`.
- **Ratio**: 16:9 base (`1280x720`). Keep long English/Korean mixed titles inside a centered safe zone.
- **Typography**: one strong headline, one smaller series line (`Building Mnemo in Public #N`). Avoid more than 2 lines in the headline.
- **Reuse vs custom**:
  - Reuse the template for weekly devlogs and architecture notes.
  - Use a custom image only for major launch, product demo, or visually meaningful graph/UI posts.
