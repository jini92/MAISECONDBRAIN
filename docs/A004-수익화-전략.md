## 1. 포지셔닝: Mnemo = BOT Suite의 "뇌"

```
MAISECONDBRAIN (Mnemo)
├── OpenClaw 스킬 (mnemo) → 모든 MAI 프로젝트에 적용
├── Obsidian 플러그인 → 커뮤니티 200만+ 사용자
├── clawhub 스킬 → OpenClaw 생태계 배포
└── BOTMEMO 앱 → BOT Suite #7
         │
    ┌────┴────────────────────────────┐
    ▼                                 ▼
BOT Suite (앱 시리즈)            MAI 프로젝트 전체
├── BOTALKS (음성대화)           ├── MAIOSS (보안)
├── BOTCON (로컬 예약)           ├── MAIBEAUTY (화장품)
├── BOTTOK (TikTok AI)           ├── MAIAX (제조)
├── BOTTUTOR (AI 교육)           ├── MAISTAR7 (인력매칭)
└── BOTMEMO (세컨드브레인) ★      └── MAIBOT (메인)
```

Mnemo는 독립 제품이면서 동시에 다른 BOT들의 **지식 기반 레이어**.
다른 봇들이 기능을 수행하면, Mnemo가 그 활동을 지식으로 연결·축적·추론한다.

## 4. BOT Suite 시너지 상세

### MAICON(BOTCON) 패턴 = Mnemo(BOTMEMO) 패턴

```
MAICON 패턴:                    Mnemo 패턴:
독립 MVP (텔레그램 봇)          독립 MVP (CLI + API)
    ↓                               ↓
clawhub 스킬 (botcon)           clawhub 스킬 (mnemo)
    ↓                               ↓
BOT Suite 편입 (BOTCON 앱)      BOT Suite 편입 (BOTMEMO 앱)
    ↓                               ↓
앱스토어 배포                    앱스토어 + Obsidian 플러그인
```

### Mnemo가 다른 BOT을 강화하는 구조

| BOT | Mnemo 연동 | 효과 |
|-----|-----------|------|
| **BOTALKS** | 대화 히스토리 → 지식그래프 | 맥락 기억 강화, "지난번에 얘기한 그거" 추론 |
| **BOTCON** | 예약 패턴/선호도 → 그래프 | "지난번 좋았던 곳과 비슷한 데" 개인화 추천 |
| **BOTTOK** | TikTok 댓글 분석 → 지식 축적 | 트렌드 패턴 발견, 크리에이터 관계 매핑 |
| **BOTTUTOR** | 학습 진도 → 그래프 관리 | 망각 곡선 기반 복습, 약점 연결 분석 |

### 크로스 BOT 데이터 흐름

```
BOTALKS 대화 → Mnemo 지식그래프 → BOTCON 추천 개선
BOTTOK 분석  → Mnemo 트렌드 축적 → MAIBEAUTY 마케팅 인사이트
BOTTUTOR 학습 → Mnemo 진도 관리 → BOTALKS 복습 알림
```

## 5. GTM (Go-to-Market) 전략

### Phase 1: MAIBOT 내부 적용 (0-4주)

```
MAIBOT에 Mnemo 적용 → 자체 검증
    ↓ 결과물: 작동하는 GraphRAG + 사용 사례
```

- 비용: $0
- 목표: 기술 검증 + 사용 사례 축적

### Phase 2: Obsidian 플러그인 배포 (4-8주)

```
Obsidian 커뮤니티 플러그인 등록 → 초기 사용자 확보
    ↓ 무료 기본 + 프리미엄 업셀
```

- 타겟: Obsidian 커뮤니티 (Reddit, Discord, Forum)
- 마케팅: 사용 사례 영상, 옵시디언 한국 커뮤니티

### Phase 3: clawhub + BOTMEMO (8-12주)

```
clawhub 스킬 배포 + BOTMEMO 앱 출시
    ↓ OpenClaw 생태계 + BOT Suite 시너지
```

- clawhub: OpenClaw 사용자 즉시 접근
- BOTMEMO: App Store/Google Play (BOTALKS 인프라 재사용)

### Phase 4: 교육 + Cloud (12주+)

```
온라인 코스 출시 + Mnemo Cloud 베타
    ↓ 수강생 → 유료 전환 퍼널
```
