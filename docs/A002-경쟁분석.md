## 2. 주요 경쟁 제품 분석

### 2.1 Notion AI

| 항목 | 내용 |
|------|------|
| **가격** | 무료~$10/월 (AI 추가 $10/월) |
| **핵심 기능** | 블록 에디터, 데이터베이스, AI Q&A, AI 요약 |
| **지식그래프** | ❌ 없음 (관계형 DB만) |
| **AI 검색** | 벡터 기반 시맨틱 검색 |
| **데이터 소유권** | ❌ 클라우드 전용 |
| **강점** | 대중적 UX, 팀 협업, 풍부한 템플릿 |
| **약점** | 지식 간 관계 추론 불가, 데이터 잠금, 오프라인 제한 |
